自己的个人博客并在Github上贡献过代码，所以为了赶上需求，我也搭建个个人博客，没事写写开发过程
中的一些心的，也算做个总结，今天就记载并分享一下搭建博客的整个步骤，毕竟自己刚搭建完成，也算
有点经验哈，话不多说，开始：

安装 Node.js

因为Hexo基于Node.js的，那么我们搭建博客网站首先需要安装Node.js环境。 Node.js 是一个基于 
Chrome V8 引擎的 JavaScript 运行环境，可以在非浏览器环境下，解释运行 JS 代码。 
去官网（http://node.org）进行下载最新的nodejs版本，下载后一路默认就行，实在不会就找
